# EN
## qb-jobChat
**job/emergency chat for fivem using qb**

### Command
- ``/jobc [message]`` Messages to people who have the same job

### Setup
Just add this script to your resources folder and write ``ensure qb-jobChat`` to your
server.cfg
- This mod uses qb-Core
